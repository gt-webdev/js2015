Closures


function mkadd(num) {
    return function(a) {
        return a + num;
    };
    num = 1;
}
var addFive = mkadd(5);
alert(addFive(3));

This is a closure because num is not an explicit parameter in the method. And it's value is being closed over by the function definition.

Understanding Type Coersion

Javascripts type coersion system makes excessive use of the following methods:

  • toString()
  • valueOf()

Addition

  1. If either operand is an object convert it to a primative.
    1. If valueOf is a primitve use that.
      (most objects don't return a primative)
    2. Otherwise call toString
  2. If either operand is a string, convert the other to a string.
  3. Otherwise convert primitive operands to numbers, and add them together.
